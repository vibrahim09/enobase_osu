AI-Powered Formula Engine

Enabling natural language formula creation for Enobase's Enterprise Resource Planning platform

Capstone Project - Spring 2025

Project Overview

Our team developed an innovative formula engine that integrates with Enobase's ERP platform, allowing users to create complex business formulas using natural language queries. The AI agent translates these queries into functional formula UI components, making business logic creation accessible to non-technical users.

Problem Statement

Traditional ERP systems require technical expertise to create custom formulas and business logic. This creates a barrier for business users who understand their processes but lack programming skills.

Our Solution

We built a natural language formula engine that allows users to describe their business logic in plain English, which is then automatically converted into functional formulas with an intuitive UI.

Target Audience

Business analysts, operations managers, and non-technical users who need to create custom business logic without programming expertise.

Key Features

Natural Language Processing

Users can describe complex business logic in plain English, which our system converts into executable formulas.

Dynamic UI Generation

Automatically creates intuitive, editable formula UI components that update in real-time.

AI-Powered Translation

Leverages advanced LLM integration to understand context and intent from user queries.

Database-Level Execution

Late-project implementation of PLV8 allows formula execution directly within the database for improved performance.

Seamless ERP Integration

Designed specifically to integrate with Enobase's existing ERP infrastructure and workflow.

Real-Time Preview

Users can see immediate results as they build and modify formulas through the interface.

Technical Implementation

Built with modern technologies for scalability and performance

Next.js React TypeScript Node.js LLM Integration PLV8 PostgreSQL

Architecture Overview

The system consists of a React frontend that communicates with a Next.js backend, which processes natural language through LLM APIs and generates formula components dynamically.

Key Challenges

Translating ambiguous natural language into precise business logic, ensuring formula accuracy, and maintaining performance with real-time updates.

Solutions

Implemented robust prompt engineering, validation layers, and introduced database-level formula execution using PLV8 for enhanced performance.
